Python Program: Compare Two Numbers
Description

This Python program takes two numbers as input from the user and checks whether the first number is greater than or equal to the second number.

It prints True if the first number is greater than or equal to the second number, otherwise it prints False.

Code
a = int(input("enter first number: "))
b = int(input("enter second number: "))
print(a >= b)
How It Works

The program asks the user to enter the first number.

The program asks the user to enter the second number.

Both inputs are converted to integers using int().

The comparison operator >= checks if a is greater than or equal to b.

The result (True or False) is printed.

Example

Input

enter first number: 10
enter second number: 5

Output

True

Another Example

Input

enter first number: 3
enter second number: 7

Output

False
Concepts Used

input() function

Type conversion using int()

Comparison operator >=

print() function
